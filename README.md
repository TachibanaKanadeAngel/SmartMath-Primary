# SmartMath-Primary
 核心技术亮点全学段智能生成引擎：核心由 MathGenerator 类驱动，支持从基础加减乘除到高年级的小数、分数、比例、简易方程及几何运算。数形结合可视化：集成 Canvas 几何渲染器，可根据题目参数（半径、边长等）实时绘制对应的几何图形。自适应难度与去重算法：系统能根据连续答对情况自动调整难度上限，并利用 GCD 约分算法识别并过滤等价题目（如识别 $1/2$ 与 $2/4$ 为重复），确保练习的高效性。工业级 PWA 体验：采用现代 Service Worker 技术（v2 缓存），支持离线使用、一键“添加到主屏幕”以及主动版本更新通知。多维度能力画像：内置多账号存档管理，利用 ECharts 将历史错题数据转化为“计算、逻辑、空间、应用、数感”五维能力雷达图，实现精准补漏。
